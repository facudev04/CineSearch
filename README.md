# 🎬 CineSearch - Movie App 2026

CineSearch es una aplicación web de vanguardia diseñada para entusiastas del cine. Permite explorar las últimas tendencias globales, realizar búsquedas precisas y sumergirse en una experiencia visual inmersiva para conocer los detalles de cada producción.

## ✨ Características Principales

* **Exploración Dinámica**: Listado actualizado de películas en tendencia ("Trending Now") y títulos populares.
* **Búsqueda Inteligente**: Motor de búsqueda funcional que consulta la base de datos de TMDB en tiempo real.
* **Interfaz Inmersiva (Estilo Hoppers/Scream)**:
    * **Vista Monumental**: Al seleccionar una película, la interfaz se transforma con un póster gigante y fondos atmosféricos.
    * **Datos Técnicos**: Visualización clara de rating (puntuación), año de estreno, duración y géneros.
    * **Sección de Resumen**: Apartado dedicado ("Overview") para leer la sinopsis completa de cada obra.
* **Diseño Premium**: Estética oscura con bordes ultra redondeados (45px-60px) y espaciado generoso para una navegación prolija.

## 🛠️ Tecnologías Utilizadas

Este proyecto utiliza las herramientas más modernas del ecosistema React:

* **React 19**: Biblioteca principal para la gestión de estados y componentes.
* **Vite**: Entorno de desarrollo rápido y optimizado para el despliegue.
* **Tailwind CSS v4**: Framework de utilidades para un diseño responsivo y estilizado sin bordes innecesarios.
* **TMDB API**: Integración con la API oficial de The Movie Database para obtener datos actualizados.

## 🚀 Instalación y Uso

Sigue estos pasos para ejecutar el proyecto en tu máquina local:

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/facudev04/cinesearch-app.git](https://github.com/facudev04/cinesearch-app.git)
    cd cinesearch-app
    ```

2.  **Instalar las dependencias:**
    ```bash
    npm install
    ```

3.  **Configurar la API Key:**
    Asegúrate de que tu clave de API (`328eb2b2f9105533279be240dee8a8da`) esté configurada correctamente en el archivo `App.jsx`.

4.  **Iniciar el entorno de desarrollo:**
    ```bash
    npm run dev
    ```

# 🎬 CineSearch - Movie App 2026

CineSearch is a cutting-edge web application designed for cinema enthusiasts. It allows users to explore the latest global trends, perform precise searches, and immerse themselves in an interactive visual experience to discover details about every production.

## ✨ Key Features

* **Dynamic Exploration**: Real-time updated list of trending movies ("Trending Now") and popular titles.
* **Smart Search**: A functional search engine that queries the TMDB database in real-time.
* **Immersive Interface (Hoppers/Scream Style)**:
    * **Monumental View**: When a movie is selected, the interface transforms with a giant poster and atmospheric backgrounds.
    * **Technical Data**: Clear display of ratings, release year, duration, and genres.
    * **Overview Section**: A dedicated space to read the full synopsis of each work.
* **Premium Design**: Dark aesthetic with ultra-rounded edges (45px-60px) and generous spacing for a clean navigation experience.

## 🛠️ Technologies Used

This project utilizes the most modern tools in the React ecosystem:

* **React 19**: Core library for state and component management.
* **Vite**: Fast and optimized development environment for deployment.
* **Tailwind CSS v4**: Utility-first framework for responsive and stylized design without unnecessary borders.
* **TMDB API**: Integration with the official The Movie Database API to fetch up-to-date information.

## 🚀 Installation and Usage

Follow these steps to run the project on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/facudev04/cinesearch-app.git](https://github.com/facudev04/cinesearch-app.git)
    cd cinesearch-app
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Configure the API Key:**
    Ensure your API key (`328eb2b2f9105533279be240dee8a8da`) is correctly configured in the `App.jsx` file.

4.  **Start the development environment:**
    ```bash
    npm run dev
    ```

## 📸 Design Preview

The application stands out for its cinematographic approach, using large-scale typography for titles and a compact "cards" system with massive separation (gap-16) for a superior user experience.
